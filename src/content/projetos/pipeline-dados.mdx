## Contexto

A área de negócio precisava consolidar dados dispersos para análises estratégicas e indicadores executivos confiáveis.

## Desafio

- Múltiplas fontes heterogêneas  
- Dados inconsistentes  
- Ausência de governança e padronização  

## Solução

- Estruturação de Data Lake em camadas (raw, refined, curated)  
- Modelagem dimensional para BI  
- Orquestração de pipelines com Airflow  
- Controles automatizados de qualidade e rastreabilidade  

## Impacto

- Redução de ~70% no tempo de geração de relatórios  
- Dados confiáveis e auditáveis  
- Base sólida para decisões executivas com maior precisão
